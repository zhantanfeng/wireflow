# client

client wrappers wireguard, and exchange client info to each other from signaling server, register to control server,
watching the peer's event, such as ADD/DELETE implemented by grpc.

## Quick Start
```shell
linkany up
```

### using log, DEBUG/INFO/WARN/ERROR
```shell
linkany up --log-level=INFO
```
short as:
```shell
linkany up -l INFO
```
